##  Overview

Easenetics is an Android application designed specifically for older adults to learn essential digital skills in an accessible, engaging way. The app combines structured lessons, interactive quizzes, AI assistance, and voice features to create a comprehensive learning experience.

##  Features

###  Core Learning
- **Interactive Lessons**: Step-by-step digital literacy content
- **Embedded Quizzes**: Knowledge checks with instant feedback
- **Progress Tracking**: Visual progress indicators and completion tracking

###  Smart Assistance
- **AI Chatbot**: Context-aware digital literacy support
- **Voice Input**: Speak to the assistant instead of typing
- **Lesson Recommendations**: Personalized learning suggestions

###  Progress & Motivation
- **Achievement System**: Unlockable goals and milestones
- **Learning Analytics**: Comprehensive statistics and metrics
- **Weekly Progress**: Track learning trends over time

###  Accessibility
- **Text-to-Speech**: Full lesson narration
- **Voice Commands**: Hands-free interaction
- **Senior-Friendly UI**: Large text, high contrast, simple navigation

###  Security
- **Firebase Authentication**: Secure user accounts
- **Multi-Factor Authentication**: Enhanced security options
- **Profile Management**: Personal settings and preferences

##  Technical Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Architecture**: MVVM (Model-View-ViewModel)
- **Backend**: Firebase (Auth & Firestore)
- **Async**: Kotlin Coroutines & Flow
- **Voice**: Android TextToSpeech & SpeechRecognizer
